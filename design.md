# SIM卡充值提醒 - 界面设计文档

## 概述

一款简洁实用的跨平台移动应用，帮助全球异地使用电话卡的用户管理充值周期，避免因忘记充值而丢失号码。

## 屏幕列表

### 1. 登录页 (Login Screen)
- Manus OAuth 一键登录按钮（支持邮箱/手机号）
- 应用 Logo + 简短描述
- 简洁大气的欢迎页面

### 2. 首页 - 卡片列表 (Home Screen)
- 顶部：APP标题 + 用户头像
- 卡片列表（FlatList），每张卡片显示：
  - 国家/地区旗帜 + 运营商名称
  - 电话号码（部分隐藏）
  - 距离下次充值的天数（醒目显示）
  - 充值状态标签（正常/即将到期/已过期）
- 右下角浮动按钮：添加新卡片
- 空状态：引导用户添加第一张卡片

### 3. 添加/编辑卡片页 (Add/Edit Card Screen)
- 表单字段：
  - 国家/地区选择器（带搜索）
  - 运营商名称（文本输入）
  - 电话号码（文本输入）
  - 充值周期（天数）
  - 上次充值日期（日期选择器）
  - 提醒设置：提前7天/3天/1天（多选开关）
  - 备注（可选）
- 保存/取消按钮

### 4. 卡片详情页 (Card Detail Screen)
- 完整卡片信息展示
- 充值倒计时（大字显示）
- "确认已充值"按钮（重置计时器）
- 充值历史记录
- 编辑/删除操作

### 5. 设置页 (Settings Tab)
- 用户信息
- 通知设置（全局开关）
- 关于应用
- 退出登录

## 关键用户流程

### 流程1：首次使用
1. 打开APP → 登录页
2. 点击"一键登录" → Manus OAuth认证
3. 认证成功 → 跳转首页（空状态）
4. 点击"添加卡片" → 填写信息 → 保存
5. 返回首页看到卡片列表

### 流程2：收到提醒并确认充值
1. 收到本地推送通知 "您的 [运营商] [号码] 将在3天后到期"
2. 点击通知 → 打开APP卡片详情页
3. 用户去充值后 → 点击"确认已充值"
4. 系统重置倒计时，停止重复提醒

### 流程3：日常查看
1. 打开APP → 首页卡片列表
2. 一目了然看到所有卡片状态
3. 红色=已过期，黄色=即将到期，绿色=正常

## 数据模型

### SIM Card
- id: 唯一标识
- userId: 用户ID
- country: 国家/地区代码
- countryName: 国家/地区名称
- carrier: 运营商名称
- phoneNumber: 电话号码
- rechargeCycleDays: 充值周期（天）
- lastRechargeDate: 上次充值日期
- remindDays: 提醒天数数组 [7, 3, 1]
- isConfirmed: 是否已确认充值（当前周期）
- note: 备注
- createdAt: 创建时间
- updatedAt: 更新时间

### Recharge History
- id: 唯一标识
- cardId: 关联卡片ID
- rechargeDate: 充值日期
- createdAt: 创建时间
